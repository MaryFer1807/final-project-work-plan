Proyecto Final
Descripción

Repositorio con el desarrollo completo del proyecto final. Incluye el análisis exploratorio de datos, el plan de trabajo, el entrenamiento y evaluación del modelo, así como el informe final de resultados.

Objetivo

Desarrollar una solución de machine learning que cumpla con las condiciones del proyecto, manteniendo una estructura clara, código limpio y una evaluación correcta del modelo.

Contenido

Análisis exploratorio de datos (EDA)

Plan de trabajo

Código de la solución

Entrenamiento y prueba del modelo

Evaluación del modelo

Informe final con conclusiones

Estructura

El proyecto se presenta en un Jupyter Notebook con secciones claramente definidas para el plan de trabajo, el desarrollo de la solución y el informe final.
